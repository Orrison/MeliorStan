# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

Please don't publish details in a public issue or PR, report the vulnerability [here](https://github.com/Orrison/MeliorStan/security/advisories/new).

A maintainer will respond to the report as soon as possible. Typically within 5-10 business days.
